Styling Text

Common font combinations can be found at W3Schools.

When using font-family, you are relying on the user`s computer to have it installed, so a couple of them are added. At least it is important to select if serif(has curves at end) or sans-serif(simple) is wanted.

Color can be selected in hexadecimal value. First two digits are red, then green, then blue: 0000ff

Font-style chooses if you want words in italic or normal

Font weight it how bold you want it, goes from 0-900/bold.

Font-size give the pixels. The default is 16px

Text-transform allows you to change the text: capitalize, lower case, etc.

Text-align allows to put it somewhere in the page.

Relative Font-Sizing

There are two units of relative measurement, % and em. % increases with relation to the default. em multiplies the size by the number next to it, so a 16px at 2em is 32px. In the follwing example, the relative sizes take place and can be combined.
