Character Entity References

There are characters used for HTML that are also used for text, we need to help the computer differentiate between them. The main characters are: < (&lt), > (&gt), & (&amp)

Other entity references are: ©, ", or if you want a part of text to break all together or not at all, using &nbsp: "victory nor defeat"

Finally, to comment a part of code use <!--, close with -->
