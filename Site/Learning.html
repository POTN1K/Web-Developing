GitHub

Update files to repository

Open cmd on your corresponding folder (Site).

  • git status: Used to see what needs to be uploaded/changed
  • git add . : By using "." you add everything
  • git commit -m "message"
  • git push

See website

On the repository, go to configuration and find the website, by changing the url you can access the different documents uploaded.


Browser Sync

In the corresponding folder (Site), open cmd and run the following command:

browser-sync start --server --directory --files "*"
